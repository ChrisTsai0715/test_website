NotifyHub API 文档

NotifyHub 是一站式消息通知服务平台，支持短信发送、邮件发送、Push 推送、语音通知、批量多渠道发送等功能。提供 HTTP API、Serverless、Webhook 及 SDK 多种接入方式，内置 AI 智能通知渠道选择能力。

  • 短信通知 - /sms/send
  • 邮件通知 - /email/send
  • Push 推送 - /push/send
  • 语音通知 - /voice/call
  • 批量多渠道发送 - /batch/send
  • 消息状态查询 - /message/status
  • AI 智能通知 - /smart/notify
  • 消息模板管理 - /template/list
  • Webhook 接入
  • SDK 接入
